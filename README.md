# HLS_LiDAR repository
This is a repository containing scripts used for a publication by Léa Enguehard et al., 2026. (under review)

**Exploring the potential of Harmonized Landsat-Sentinel-2 in predicting boreal forest structure from UAV-LiDAR data in Northwestern America.** Enguehard, Léa; Kruse, Stefan;  Hänsch, Ronny; Herzschuh, Ulrike; Panda, Santosh; Heim, Birgit. _Under Review (2026)_

***

# Purpose of HLS_LiDAR repository
Provide scripts used to:
-  Collect HLS time-series
-  Aggregate UAV-Lidar data with HLS and prepare benchmarked dataset
-  Produce the publication's figures and data analyses


**Reference to dataset**

Enguehard, Léa; Kruse, Stefan; Farkas, Luca; Gloy, Josias; Heim, Birgit (2025): Harmonized Landsat and Sentinel-2 tiles labeled with UAV-LiDAR-derived forest structural attributes in Alaska and Northwest Canada [dataset]. PANGAEA, https://doi.org/10.1594/PANGAEA.983738
